## 一、安装 vue
    
    # 查看版本
    npm -v
    2.3.0

    #升级 npm
    cnpm install npm -g

    # 升级或安装 cnpm
    npm install cnpm -g
    
    # 最新稳定版
    npm install vue

## 二、命令行工具 vue-cli

    vue-cil是vue的脚手架工具
    # 全局安装 vue-cli
    cnpm install --global vue-cli
    
    # 创建一个基于 webpack 模板的新项目
    vue init webpack my-project
