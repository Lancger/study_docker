# 一、固定时间点
```
使用 el-time-select 标签，分别通过star、end和step指定可选的起始时间、结束时间和步长
```
```
<template>
  <div>
  <el-time-select
    v-model="value"
    :picker-options="{
      start: '08:30',
      step: '00:15',
      end: '18:30'
    }"
    placeholder="选择时间">
  </el-time-select>
  </div>
</template>

<script>
export default {
  data () {
    return {
      value: ''
    }
  }
}
</script>
```
