# 二、Html两大元素
```
块元素总是独占一行，而内联元素和相邻的元素在同一行，如果一行内行宽不够时，才会被挤到另一行

这两种元素的排列方式是在没有特殊属性的干扰下，在正常的标准流中所表现出来正常的特性
```
