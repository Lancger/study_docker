# 一、固定时间点
```
使用 el-time-select 标签，分别通过star、end和step指定可选的起始时间、结束时间和步长
```
```
<template>
  <div>
  <el-time-select
    v-model="value"
    :picker-options="{
      start: '08:30',
      step: '00:15',
      end: '18:30'
    }"
    placeholder="选择时间">
  </el-time-select>
  </div>
</template>

<script>
export default {
  data () {
    return {
      value: ''
    }
  }
}
</script>
```

# 二、任意时间点
```
可以选择任意时间

使用 el-time-picker 标签，通过selectableRange限制可选时间范围。提供了两种交互方式：默认情况下通过鼠标滚轮进行选择，打开arrow-control属性则通过界面上的箭头进行选择。

```
```
<template>
<div>
  <el-time-picker
    v-model="value1"
    :picker-options="{
      selectableRange: '18:30:00 - 20:30:00'
    }"
    placeholder="任意时间点">
  </el-time-picker>
  <el-time-picker
    arrow-control
    v-model="value2"
    :picker-options="{
      selectableRange: '18:30:00 - 20:30:00'
    }"
    placeholder="任意时间点">
  </el-time-picker>
</div>
</template>

<script>
  export default {
    data() {
      return {
        value1: new Date(2016, 9, 10, 9, 40),
        value2: new Date(2016, 9, 10, 18, 40)
      };
    }
  }
</script>
```
